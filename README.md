# streamline-cms
The simple and intuitive headless CMS.

# For Execute on Local

## Sytem Requirements

* Docker
* Port 18000 on local. (see docker-compose.yml and configure if you needed.)

## Local Setup

```bash
docker compose up -d
```

## Show API Specifications.

* see: http://localhost:18000/docs/

# For Development

## Run tests

```bash
pytest -v
```
